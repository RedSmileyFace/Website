NPC Script

  • Created By: Red =)

Have you ever wanted to customize your NPCs in skript but don't know where to start? We know the struggle it can be so we created an easy to use skript to allow you to customize NPCs with messages, commands, and cooldowns!

Features:

  1. Easily Set NPC Chat Format
  2. Customize Messages
  3. Set Message Delays
  4. Set Commands per NPC
  5. Set Messages per NPC
  6. Set Chat Colors per NPC

Setup Steps:

  1. Download all the Dependencies from Here !
  2. Install the Dependencies onto your Bukkit / Spigot / Paper Server!
  3. Restart your Server!
  4. Download the NPC Configuration Script from Here !
  5. Install it into your Scripts Folder!
  6. Reload the Script or Restart your Server!

Commands:

  1. /npcsk [Reload / Info]
  2. Reloads the NPC System, View your Current Version through In-Game commands!
  3. Permission Node: command.npcsk

Tutorials:

{

   "Name": "ExampleNPC",

   "ChatColor": "&c",

   "Messages": [

       "Hey &c[PLAYER]&r, I hope you have a fun time!!!"

   ],

   "Commands": [

       "say Hey!"

   ]

}

Navigate to the plugins/iCodeDevelopment/NPCS/ Directory on your Server

 

1) Within the NPCS Folder you will find the config.json file that controls

the whole skript along with a directory named NPCS-DATA

2) Inside the NPC-Data Folder you will create a npc-name.json File

3) Copy and Paste the Example NPC File into your new NPC File!

4) Customize it to your liking!

"Name":   Specifies the Name that the NPC will use in chat.

          (Supports Color Codes + Hex Colors)

"ChatColor":   Specifies teh chat color replacement in your Message

               Format set in Config.

               (Supports Color Codes + Hex Colors)

"Messages":   Is a JSON Stage in which you are able to specify unlimited

              amounts of messages that will be sent to the Player.

              (Supports Color Codes + Hex Colors)

"Commands":   Is a JSON Stage in which you are able to specify

              unlimited amounts of commands beneath this stage.

[Player] - Replaced with Player's Name

[CHAT COLOR] - Replaced with the Color Code of your Choice

[NPC NAME] - Replaced with the Name of the NPC

[MESSAGE] - Replaced with the Message Specified
