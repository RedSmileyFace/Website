Play Minecraft Lucky Block Challenge Games vs Aliens! Instead of raiding Area51 how about fight Aliens in Minecraft!?!? This all new Challenge Games is the 3rd return of the Challenge Games series for PopularMMOs!

Setup Steps:

  1. Download and Install Forge 1.12.2 (14.23.5.2847) onto your Minecraft Client!
  2. Open your Minecraft Launcher
  3. Navigate to the TAB labelled Installations
  4. Click "+ New" to create a New Installation
  5. Set the Version to release 1.12.2-forge1.12.2-14.23.5.2847
  6. If you have not already, Drag and Drop the Folder from the ZIP File onto your Computer, then set the Game Directory to that Folder!
  7. Click on Create in the bottom Right Hand Corner!
  8. Launch your New Minecraft Installation
