Lobby Script

  • Created By: Red =)

Have you ever wanted simple lobby commands but don't want to spend the time making them? We know there are tons of scripts out there but are they fully customization in a config file? I think not!

Features:

  1. Easily Set Custom Join & Leave Messages
  2. Customize Messages
  3. Set Join Coordinates
  4. Set Private Messages on Join
  5. Set Titles and Subtitles on Join

Setup Steps:

  1. Download all the Dependencies from Here !
  2. Install the Dependencies onto your Bukkit / Spigot / Paper Server!
  3. Restart your Server!
  4. Download the Lobby Configuration Script from Here !
  5. Install it into your Scripts Folder!
  6. Reload the Script or Restart your Server!

Commands:

  1. /lobby [Reload / Info]
  2. Reload the Lobby System, View your Current Version through In-Game commands!
  3. General Permission Node: command.lobby
  4. Admin Permission Node: command.lobby.admin
  1. /sendhub [Player]
  2. Sends the Specified Player to the Server Hub!
  3. Permission Node: command.sendhub
  1. /sendallhub
  2. Sends Everyone to the Server Hub!
  3. Permission Node: command.sendallhub
