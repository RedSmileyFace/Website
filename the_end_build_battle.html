The End - Build Battle

  • Created By: Red =)
  • Distributed By: RedSmileyFace
  • Comissioned By: Bendie

Have you ever wanted to host a Minecraft Build Battle with an End theme? Well, now you can! Launch your own vanilla minecraft server using this download which includes everything from custom scripts, to builds themed around Bendie!

Setup Steps:

  1. Download and Install Vanilla 1.16.5 onto your Minecraft Client!
  2. Open your Minecraft Launcher
  3. Navigate to the TAB labelled Installations
  4. Click "+ New" to create a New Installation
  5. Set the Version to Minecraft Vanilla 1.16.5
  6. If you have not already, Drag and Drop the Folder from the ZIP File onto your Computer!
  7. Run the Start.bat File found in the Directory!
  8. Launch your Minecraft Client
  9. Type localhost as the Server Address!

Installation Recommendation:

Some maps utilize a feature in Minecraft Java Edition to constantly update your client every tick!

I recomend designating more RAM / Memory to Minecraft in order to ensure the best gaming experience possible.
