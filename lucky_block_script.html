Have you ever wanted to create your very own lucky block but don't know where to start off? We have guides on how you can get started to create your very own Lucky Block!

Features:

  1. Custom Resource Pack
  2. Limitless Lucky Block Creations
  3. Limitless Drop Files
  4. Customization Debug Menu
  5. Custom Messages
  6. Auto Updater
  7. Custom Discord Bot Creation

Setup Steps:

  1. Download all the Dependencies from Here !
  2. Install the Dependencies onto your Bukkit / Spigot / Paper Server!
  3. Restart your Server!
  4. Download the Lucky Block Configuration Script from Here !
  5. Install it into your Scripts Folder!
  6. Reload the Script or Restart your Server!

Commands:

  1. /luckyblock [Reload / Info / Update]
  2. Reload the Lucky Block System, View your Current Version, or Update the System through In-Game commands!
  3. Permission Node: command.luckyblock
  1. /debugmenu
  2. Opens the In-Game Debug Menu!
  3. Permission Node: command.debugmenu

1)   Create a Folder named MY-LUCKY-BLOCK replacing spaces with - with

     all Capital Letters.

2)   Within your New Lucky Block Folder there can be Three Categories:

       - Unlucky Lucky Block

       - Normal Lucky Block

       - Super Lucky Block

     Theses are there own blocks and are opened separately from each other.

3)   Within each Lucky Block Type you have Three Tiers:

       - High Drop Value

       - Medium Drop Value

       - Low Drop Value

     Each Tier is its own Folder in which you will place drop files inside.

4)   Create Drop Files within each Category by naming the file drop_#.json

     then follow the tutorial below to edit Drop Files!

"key":   Specifies what Drop Number corresponds with your drops.

         It is used to identify when the randomized drop is selected in-game.

"author":  Specifies the creator of the drop.

"Items":   Is a JSON Stage which you are able to specify unlimited

           amounts of items beneath this stage.

"GRENADE_LAUNCHER":   Is an example of an ID used on an Item. Beneath this

                      line you will find the attributes that are put on that drop.

"Name":   Specifies the Custom Name of the Item using Minecraft Color Codes.

          (Supports Color Codes + Hex Colors)

"Lore":   Specifies the Custom Lore of the Item using Minecraft Color Codes.

          (Supports Color Codes + Hex Colors)

"Material":   Specifies the Item Type, Amount, and its Custom NBT Data.

"Commands":   Is a JSON Stage which you are able to specify unlimited

              amounts of commands beneath this stage by specifying it into 1 of the

              following sections.

              (Player or Console)

"Player":   Is a section that is below the Commands Stage, this section

            allows you to run commands as the player. Any command

            specified will be ran through the player as OP.

            (Return Messages will still appear if applicable.)

"Console":   Is a section that is below the Commands Stage, this section

             allows you to run commands through the Server Console.

"SendMessage":    Is a JSON Stage which you are able to specify unlimited

                  amounts of messages that will be sent to the Player who broke the Lucky Block.

                  (Supports Color Codes + Hex Colors)

"BroadcastMessage":   Is a JSON Stage which you are able to specify unlimited

                      amounts of messages that will be sent to all the player's within the Arena of

                      the player who broke the block.

                      (Supports Color Codes + Hex Colors)

"Structure":   Is a JSON Stage which you are able to specify a Structure using

               the Vanilla Minecraft Structure Blocks to Save the Structure.

"Hologram":   Is a JSON Stage which will spawn a Hologram Above the Lucky Blocks

              Location when broken.

              (Supports Color Codes + Hex Colors)

[Player] - Replaced with Player's Name

[PlayerUUID] - Replaced with Player's UUID

[BlockLocation] - Replaced with the Lucky Block's Location

[PlayerLocation] - Replaced with Player's Location

[Random](#-#) - Replaced with a Random Number

[RandomList](First,Second,Third) - Replaced with a Random Input from the List
